# RAG Chatbot (Retrieval-Augmented Generation)

This project is a simple AI chatbot that can answer questions based on custom documents (PDF/text files).

## Features
- Upload documents (PDF or text)
- Split text into chunks
- Store embeddings in vector database
- Retrieve relevant context for user questions
- Generate answers using an LLM

## Tech Stack
- Python
- OpenAI API (or open-source LLM)
- LangChain (optional)
- FAISS / ChromaDB
- Streamlit (UI)
